# Scripts

This folder contains Python scripts for reusable functions, utilities, and automation in the News Sentiment Stock Prediction project.

## Contents
- **Data Processing Scripts**: Cleaning and preparing financial news and stock datasets
- **Technical Indicators Functions**: Calculating MA, RSI, MACD using TA-Lib and PyNance
- **Sentiment Analysis Utilities**: Functions for scoring news headlines
- **Helper Functions**: Aggregating and aligning data by date for correlation analysis

## Usage
Run scripts from the command line or import them into notebooks:

```bash
python scripts/your_script.py
````

Scripts are modular and can be reused in multiple notebooks or workflows to maintain clean and efficient code.
