# Notebooks

This folder contains Jupyter notebooks used for analysis, visualization, and experimentation in the News Sentiment Stock Prediction project.

## Contents
- **Exploratory Data Analysis (EDA)** on financial news headlines and stock data
- **Technical Indicators** calculations using TA-Lib and PyNance (MA, RSI, MACD)
- **Sentiment Analysis** of news headlines using NLP libraries (TextBlob, NLTK)
- **Correlation Analysis** between daily sentiment scores and stock returns

## Usage
Open the notebooks in Jupyter Lab or Jupyter Notebook:

```bash
jupyter notebook notebooks/
````

Each notebook is self-contained and can be executed step by step to reproduce the analysis, visualizations, and results.
