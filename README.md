# News Sentiment Stock Prediction

## Overview
Week 1 Challenge – 10 Academy Artificial Intelligence Mastery Program.  
This project analyzes the relationship between financial news sentiment and stock price movements. It combines Exploratory Data Analysis (EDA), Natural Language Processing (NLP), and quantitative finance techniques.

## Objectives
- Perform EDA on financial news headlines and stock data
- Compute sentiment scores using NLP tools
- Apply technical indicators (MA, RSI, MACD) with TA-Lib and PyNance
- Examine correlation between news sentiment and daily stock returns
- Provide actionable insights for predictive financial analytics

## Tools & Libraries
- Python 3.x
- Pandas, NumPy, Matplotlib, Seaborn
- TA-Lib, PyNance
- NLTK, TextBlob (for sentiment analysis)
- Git & GitHub for version control
- GitHub Actions for CI/CD

## Usage
1. Clone the repository:
```bash
git clone https://github.com/your-username/news-sentiment-stock-prediction.git
````

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run Jupyter notebooks in `notebooks/` or scripts in `src/` to perform analysis.
